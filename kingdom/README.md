# 🏰 Kingdom

Welcome to the **Kingdom**! 👑  
This is the main social world of the Gatecraft server. The world is designed **exclusively for founding nations, building cities, and community life**. 

🚫 **Important Rule:** Resource mining is **strictly forbidden** in this world to keep nature untouched! Gather your resources in the Mining Colony.
